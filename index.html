Cockatrice is an open-source multiplatform software for playing card games, such as Magic: The Gathering, over a network. It is fully client-server based to prevent any kind of cheating, though it supports single-player games without a network interface as well. Both client and server are written in Qt, supporting both Qt4 and Qt5.



Getting Started

  1. Download a installation file or compile your own

  2. Install 'Cockatrice' application

  3. Run 'Oracle' tool to create a card database (or import one)

  4. Open 'Cockatrice' and start building a deck (or import one)

  5. Start playtest offline or connect to a server:
    • Woogerworks
    • Chickatrice
    • Poixen
      • ...contact us to see your public server here!
    Enjoy playing your favourite card games!
